# Coding Lab Repository

Welcome to **Coding Lab**! This repository is a hands-on learning environment to explore different programming languages, develop projects, create CLIs, APIs, and conduct case studies. The goal is to improve technical skills through challenges, experimentation, and constant practice.

## 🚀 Purpose
The **Coding Lab** was created to serve as an experimental and learning space for programming. Here, practical projects, coding challenges, and studies of fundamental concepts will be developed to enhance developer skills.

## 📌 What will you find here?
- **Practical Projects**: Small and medium-sized projects to apply learned concepts.
- **CLI Development**: Command-line tools for automation and productivity.
- **API Creation**: Exploring different approaches to building APIs.
- **Coding Challenges**: Problems and algorithms to improve programming logic.
- **Case Studies**: Analysis and solutions for real-world problems with different approaches.
- **Benchmarks**: Performance tests between different languages and methodologies.

## 🛠️ Technologies and Languages
The lab will explore multiple technologies, with a particular focus on:
- **Golang**
- **Python**
- **JavaScript/TypeScript**
- **Bash/Shell Script**
- **Docker and Kubernetes** (for deployment and application orchestration)
